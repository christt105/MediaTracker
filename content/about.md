Bienvenido a mi Media Tracker.

Esta página web es un registro de todas las películas, series y videojuegos que he consumido desde que hago seguimiento de ello.

Previamente usé Notion para hacerlo. Creé una plantilla y se podía generar una página web fácilmente, pero no me acabab de convencer. Desde que usé Obsidian, migré todo el contenido a él. El problema con Obsidian es que no tiene una opción para generar una página web fácilmente. Por eso decidí usar Hugo.

### Generación

Esta página web está generada a partir de mis notas de [Obsidian](https://obsidian.md/) usando [Hugo](https://gohugo.io/).
